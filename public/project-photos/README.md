# Project Photos

Taruh 4 foto project lu di folder ini.

Nama file default yang dipakai website:

- `project-01.svg`
- `project-02.svg`
- `project-03.svg`
- `project-04.svg`

Bisa ganti jadi `.jpg`, `.png`, atau `.webp`, tapi update path di `src/main.tsx` kalau nama/ekstensinya berubah.

Rekomendasi:
- Rasio 1:1 biar cocok di InfiniteMenu.
- Ukuran aman: 900x900 sampai 1400x1400.
- Format paling waras: `.webp` atau `.jpg`.
